# silicat_scada — SCADA-система управления дозаторами

Десктопная SCADA-система (C++/Builder, VCL) для визуализации и управления дозаторным производством силикатных изделий. Работает в паре с прошивкой контроллера [silicatPLC](https://github.com/exclusivelogin/silicatPLC).

## Назначение

Оперативное диспетчерское управление производством: отображение текущих/заданных производительностей дозаторов (известь/песок), трендовый анализ, архивирование исторических данных в Access (.mdb), управление уставками оператором.

## Архитектура системы

```
silicat_scada/
├── code/
│   ├── Unit1.cpp / .h / .dfm     # Главная форма SCADA (TWinScada): дашборд + управление
│   └── FormOpros.cpp / .h / .dfm # Форма опроса (TPACQuery): коммуникация с ПЛК
├── MyCLASS/
│   └── ClassArhiv.cpp / .hpp     # Класс архивирования — запись показаний в БД с тегами
├── project/
│   ├── Arhiv.mdb                 # База данных архива (Microsoft Access)
│   ├── dozatory_arm.bpr          # Проектный файл C++Builder
│   ├── ARHIV/                    # Файловый архив (.dat файлы по датам)
│   └── *.bmp / *.jpg             # Ресурсы интерфейса
```

## Функциональность

| Блок | Описание |
|------|---------|
| **TWinScada** | Главное окно: отображение текущих/заданных производительностей 4 дозаторов, управляющие кнопки, дашборд состояния |
| **TPACQuery** | Фоновый процесс: опрос ПЛК через Ethernet/TCP, статусы соединения (connected/disconnected/lookup) |
| **ClassArhiv** | Периодическое сохранение тегов в Access (.mdb) и файловый архив (.dat по дате/часу) |
| **Тренды** | Исторические графики показателей — фильтрация по тегам, временной диапазон через DateTimePicker |
| **Архив** | Хранение и просмотр исторических данных; до 100 активных тегов (Lic=100) |

## Стек технологий

| Технология | Назначение |
|-----------|-----------|
| C++Builder 6 (Borland/Embarcadero) | IDE и компилятор (VCL framework) |
| VCL (Visual Component Library) | GUI — TForm, TTimer, TDBGrid, TChartFX |
| Ethernet/TCP | Связь с ПЛК (ICP DAS 7188XW) |
| Microsoft Access (.mdb) | Архивная база данных (BDE/ADO) |
| DBGridEh | Расширенная таблица данных |

## Схема взаимодействия

```
Оператор (SCADA GUI)
        ↕  Ethernet TCP
ICP DAS 7188XW (silicatPLC)
        ↕  RS-485 / Modbus RTU
4 дозатора (известь × 2, песок × 2)
```

## Статусы соединения с ПЛК

| Код | Состояние |
|-----|----------|
| 1 | Ethernet подключён (clGreen) |
| 2 | Контроллер отключён (clRed) |
| 3 | LookUP — поиск соединения (clGreen) |

## Контекст

Система разработана в 2014–2015 гг. для реального производственного объекта. Полный стек: от firmware ПЛК ([silicatPLC](https://github.com/exclusivelogin/silicatPLC)) до десктопной SCADA с архивированием и трендами.
